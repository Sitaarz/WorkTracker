# WorkTracer
## MVP
Celem MVP jest zbudowanie prostego systemu do zarządzania zadaniami w postaci tablicy kanban.

### Zakres MVP
- rejestracja i logowanie użytkownika
- autoryzacja JWT
- tworzenie, edycja, usuwanie zadań
- oznaczanie statusu zadania
- filtrowanie listy zadań
- podstawowe testy jednostkowe
- deployment

### Poza MVP
- przypisywanie zadań innym użytkownikom
- komentarze
- powiadomienia email
- zaawansowane raporty
